## Local preview

If this is a Jekyll site, you can preview locally with:

```bash
bundle install
bundle exec jekyll serve
```

Or simply open the HTML files in a browser if it's a static site.

## How to edit

1. Edit files directly in this repository or clone it locally:

```bash
git clone https://github.com/rogeryeepam/rogeryeepam.github.io.git
```

2. Create a branch, make changes, and open a pull request to merge into `main`.
